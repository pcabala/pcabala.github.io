# Aizawa

**Equations** (one common form)
systems\rossler.md
\begin{aligned}
\dot{x} &= (z - b)x - d y,\\
\dot{y} &= d x + (z - b)y,\\
\dot{z} &= c + a z - \frac{z^3}{3} - (x^2 + y^2)(1 + e z) + f z x^3.
\end{aligned}
